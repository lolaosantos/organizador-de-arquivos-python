# Organizador de Arquivos com Python

Projeto desenvolvido com o objetivo de automatizar a organização de arquivos em diretórios.

<img width="615" height="676" alt="Captura de tela 2026-04-16 233244" src="https://github.com/user-attachments/assets/32472c44-b22e-46a3-bcb1-1e42f0a0cc68" />


## Funcionalidades
- Organização automática de arquivos por extensão
- Criação de pastas dinamicamente
- Redução de tarefas manuais

## Tecnologias utilizadas
- Python
- Biblioteca os
- Biblioteca shutil

## Como funciona
O sistema percorre um diretório e move os arquivos para pastas específicas de acordo com suas extensões.

##  Exemplo
Arquivos como:
- .jpg → pasta Imagens
- .pdf → pasta Documentos
- .mp4 → pasta Vídeos
